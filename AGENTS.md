# ADAD Workspace configuration and project rules.
# This file defines project rules for Lobar Union ADAD workflow.

## 📋 服務人員出勤天數精算與動態排假規範 (Attendance Calculation Rules)

1. **兩階段時間原則與兩階段操作選單**：
   - 洽談中案件與確定開始日案件隔離控制：選單提供「1. 執行操作」(不連動單純看行事曆 / 訂單匹配 / 出勤天數精算) 與「2. 訂單選擇」(預設為無)。
   - 「訂單匹配」僅於月曆展示黃底預排與 7 天預留備用期，**不顯示出勤精算面板與單日排假**；「出勤天數精算」僅適用於確定實際服務開始日 (`actual_start_date`) 案件，觸發紅底鎖定與綠底休假動態順延。

2. **綠底休假與紅底動態順延**：
   - 排定之休假/請假在行事曆上以 **綠底 🟢 (休假/請假)** 標示。
   - 不論預設休假、動態請假或國定假日休假，每增加 1 天綠底休假，後續紅底 🔴 工作日與服務結束日期自動向後順延 1 天，確保實際服務天數 100% 足額達 $N$ 天。

3. **服務方式與跨週單日排假**：
   - 支援「連續服務」、「週休1日」、「週休2日」。
   - 考慮每週休假星期可能不同，休假直接在行事曆上針對個別日期進行點選或調整，每次排假自動順延 1 天。

4. **國定假日單日獨立決策**：
   - 國定假日無雙倍薪資加給條款。
   - 支援連假期間（如端午連假 3 天）針對每一個獨立國定假日單日個體勾選是否出勤；選擇休假者結束日自動順延 1 天，選擇出勤者計為正常工作日。
